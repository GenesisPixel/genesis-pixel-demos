Canvas

¿Qué es Canvas?

HTML Canvas es un elemento que te permite dibujar gráficos bidimensionales directamente en la web usando JavaScript. Piensa en él como un lienzo en blanco: puedes trazar líneas, formas, texto, imágenes y hasta animaciones completas pixel a pixel.

A diferencia de SVG, que trata cada forma como un objeto DOM individual, Canvas opera a nivel de píxel. Esto lo hace extremadamente eficiente para gráficos que cambian constantemente, como simulaciones de partículas, juegos o efectos visuales en tiempo real.

¿Para qué sirve?

Juegos 2D:

Plataformas, puzles y juegos arcade que corren directamente en el navegador sin plugins.

Gráficos de datos:

Chart.js, D3.js y otras librerías usan Canvas para renderizar gráficos interactivos de miles de puntos.

Efectos visuales:

Fondos animados, generadores de arte procedural y filtros de imagen en tiempo real.

Editores de imagen:

Aplicaciones como Figma o Photopea usan Canvas como su superficie de renderizado principal.

Canvas vs SVG

Canvas: Renderizado por píxel. Ideal para gráficos complejos y animaciones fluidas. No tiene eventos por elemento individual.

SVG: Renderizado por objetos. Cada forma es un elemento DOM con sus propios eventos. Ideal para iconos, diagramas y gráficos interactivos.

Consejo profesional

Si necesitas interacción directa con elementos individuales (clic en una barra del gráfico), usa SVG. Si necesitas rendimiento bruto para miles de elementos simultáneos, usa Canvas.
